📖 Mode d'emploi :
  1. Choisissez votre rôle : Hôte (celui qui crée le chat) ou Invité
  2. Hôte : cliquez "Générer l'offre" → copiez le texte → envoyez-le à l'invité (WhatsApp, SMS, etc.)
  3. Invité : collez l'offre reçue → cliquez "Traiter l'offre" → copiez la réponse → envoyez-la à l'hôte
  4. Hôte : collez la réponse reçue → cliquez "Envoyer la réponse"
  5. 🎉 La connexion s'établit automatiquement ! Vous pouvez discuter.
